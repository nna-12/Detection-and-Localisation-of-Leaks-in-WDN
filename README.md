# Leak Detection and Localisation Datasets for WDN

This repository contains datasets for leak detection and localisation in Water Distribution Networks (WDN) for two areas: **K.K. Nagar (Chennai)** and **R.S. Puram (Coimbatore)** .  

## Folder Overview

- **kknagar/** and **rspuram/**
  - `Results_noisy/`
    - `Demands/` – AMR sensor data
    - `Pressures/` – Pressure sensor data
    - `Flows/` – Flow sensor data
    - `Levels/` – Level sensor data
    - `Leaks/` – Individual leak scenarios (`Leak_P*.csv`)
    - `Leakages.csv` – Leak metadata
  - `.inp` – EPANET network model
  - `.yaml` – Dataset configuration
